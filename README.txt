■ 公開手順（GitHub Pages）
1) GitHub で新しいリポジトリを作成します（Public 推奨）。
   - 例: repo 名「higashinari-100th-memory-game」

2) 3つのファイル（index.html, captions_jp.txt, README.txt）を
   リポジトリの直下（ルート）にアップロードしてコミットします。
   - GitHubのWeb画面 → 「Add file」→「Upload files」→ ドラッグ＆ドロップ → Commit

3) Settings → Pages を開きます。
   - Source: 「Deploy from a branch」
   - Branch: 「main」(または「master」) ／ Folder: 「/ (root)」
   - 「Save」を押します。

4) 数十秒～数分で公開URLが発行されます。
   - 形式: https://<ユーザー名>.github.io/<リポジトリ名>/

5) 公開URLをコピーして、QRコードやSNS告知にお使いください。
   - Canva → Apps「QR Code」でURLを入れるとQR画像が作れます。

※ 更新方法：index.html を置き換えてコミットすれば、自動で再デプロイされます。
