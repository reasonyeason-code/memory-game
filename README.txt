公開のやり方（かんたん順）
────────────────────────────────
A) Netlify Drop（ドラッグ＆ドロップでURL発行）
  1. ブラウザで「Netlify Drop」を検索して開く
  2. 本フォルダの index.html をドロップ
  3. 数秒で公開URLが発行されます（例 https://*.netlify.app）

B) GitHub Pages（無料/安定）
  1. 新規リポジトリを作成 → index.html をアップロード
  2. Settings → Pages → “Deploy from branch” を選択 → 保存
  3. 数分で公開URL（https://ユーザー名.github.io/リポジトリ名）が発行されます

C) Cloudflare Pages / Vercel（ドラッグ＆ドロップ or Git連携）
  - アカウントを作成 → index.html をアップロードするとURL発行

D) 自分のサーバー（レンタル/学内）
  - 公開ディレクトリ（public_html/ や www/ 等）に index.html をアップロード
  - 例: https://your-domain.jp/halloween/ （index.html は省略可能）
